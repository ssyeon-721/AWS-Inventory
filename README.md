# AWS-Inventory

AWS Resource Inventory Scripts

이 저장소는 AWS 계정의 주요 리소스를 빠르고 안정적으로 인벤토리(CSV) 형태로 추출하기 위한 운영 실무용 스크립트 모음입니다.
추가 바이너리(jq 등) 의존성을 최소화하고, AWS CLI + python3 기본 환경만으로 동작하도록 설계되었습니다.

---

🎯 목적 (Why this repo exists)

운영/이관/보안 점검/감사 대응 과정에서 다음과 같은 요구가 반복적으로 발생합니다.

“현재 계정에 어떤 리소스가 있는지 한 번에 보고 싶다”
“콘솔 스크린샷 말고 CSV로 정리된 증적이 필요하다”
“고객사/폐쇄망 환경이라 jq 설치가 어렵다”
“속도는 중요하지만, 운영 서버에서 안정성이 더 중요하다”

이 저장소는 위 요구를 만족하기 위해 만들어졌습니다.


🏗️ 설계 원칙
1. jq 미사용

일부 고객사/운영 서버 환경에서는 jq 설치가 불가
대신 python3 표준 라이브러리(json) 로 파싱

2. API 호출 최소화

가능한 경우 List API 1회 + 파싱
리소스별 describe 반복 호출로 인한 성능 저하 방지

3. CSV 안정성

Excel/Sheets 호환을 고려한:
  따옴표 이스케이프
  개행 제거
  배열 필드 ; join

4. 운영 친화성

실행 전 기존 파일 자동 백업
일부 리전/리소스 실패 시 전체 중단 방지 (|| true)
인벤토리 “목록 레벨” 기준 (속도 우선)
